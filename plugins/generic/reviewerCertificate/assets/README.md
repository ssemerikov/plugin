# Certificate Template Assets

This directory contains default template assets for the Reviewer Certificate Plugin.

## Background Images

Place your certificate background images in this directory. Recommended specifications:

- **Format**: JPG or PNG
- **Size**: 2100 x 2970 pixels (A4 at 300 DPI)
- **Orientation**: Portrait
- **File naming**: Use descriptive names like `certificate_background.jpg`

## Creating a Custom Background

You can create a custom certificate background using:

1. **Graphic Design Software**: Adobe Photoshop, Illustrator, GIMP, Canva
2. **Template Dimensions**: 210mm x 297mm (A4)
3. **Resolution**: 300 DPI for print quality
4. **Color Mode**: RGB for digital, CMYK for print

### Design Tips

- Leave margins of at least 15mm on all sides
- Use professional fonts and colors
- Include decorative elements like borders, seals, or watermarks
- Ensure sufficient contrast for text readability
- Test the design by generating a preview certificate

## Default Template

A default template image (`default_template.jpg`) should be placed here. If no custom background is uploaded, the plugin will generate certificates with a simple white background.

## Uploading Custom Backgrounds

Administrators can upload custom backgrounds through the plugin settings interface:

1. Navigate to Settings > Website > Plugins > Reviewer Certificate Plugin
2. Click "Settings"
3. Upload your background image in the "Background Image" field
4. Click "Save"
5. Use the "Preview Certificate" button to see the result
